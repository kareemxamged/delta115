# Capabilities: `StudentCourses.tsx`

This target file provides the following precise capabilities for course cataloging:

1. **Mass Payload Hydration**: Exclusively mounts the `courseService.getEnrolledCourses()` pipeline.
2. **Context-Aware Drilling Parameters**: Capable of launching explicit state arguments out of its operational block directly into downstream systems (`{ activeTab: 'materials' }` or `{ activeTab: 'exams' }`).
3. **Derived Real-Time Quantifiers**: Dynamically calculates the numeric sum of items inside each filtering bucket to provide real-time UI counters (`[Tab] Courses (Count)`).
4. **Boolean Property Filters**: Differentiates purely between `completed` payload objects and `enrolled` objects through the UI tab structure.
5. **Partial Text Intersections**: Cross-checks string mutations simultaneously against Subject Names and Administrative Codes (e.g. `CS101`).
6. **Phantom Data Presentations**: Generates mock metrics simulating global student progress limits such as Cumulative GPAs and Credit Progress Bars.
